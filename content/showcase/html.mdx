## Como usar

| Camada | Papel |
|---|---|
| **Visual HTML** | Demonstração visual completa e navegável do design system. |
| **MDX oficial** | Documentação pública, contratos, decisões e links canônicos. |
| **`globals.css`** | Fonte de verdade dos tokens usados pelo produto. |

## Observações

- A camada HTML é publicada em `/visual/pulso-ds/`.
- Ela preserva a estrutura relativa do pacote original.
- Não substitui a documentação MDX nem altera tokens.
- É a melhor referência visual para comparar densidade, componentes e padrões.
