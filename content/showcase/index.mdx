# Showcase

Esta seção mostra o Pulso como produto de design: não substitui a documentação,
mas ajuda a explicar regras, diretrizes, componentes e comportamentos de forma
mais visual.

Os exemplos foram reconstruídos em MDX/React com tokens atuais. Os HTMLs do Open
Design servem como referência editorial e visual, não como código importado.

## Como ler esta camada

| Regra | Implicação |
|-------|------------|
| Visual é demonstração | A fonte canônica continua nas páginas MDX e em `globals.css`. |
| Sem HTML bruto | Exemplos são reconstruídos com tokens atuais. |
| Sem novos tokens | Esta etapa não altera `src/app/globals.css`. |
| Sem runtime de produto | O Pulso segue como contrato visual e documental. |
