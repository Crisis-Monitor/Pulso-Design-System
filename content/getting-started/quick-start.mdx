# Quick Start

Este guia monta uma superfície mínima alinhada ao v3: identidade por estrutura, dados numéricos em mono, risco explícito e marcação de conteúdo assistido pela Cris.

## 1. Use tokens, não valores soltos

```tsx
export function CrisisHeader() {
  return (
    <header className="border-b bg-background px-8 py-6 text-foreground">
      <p className="font-mono text-[var(--text-micro)] uppercase tracking-[var(--tracking-kicker)] text-muted-foreground">
        Monitoramento · Crise ativa
      </p>
      <h1 className="mt-3 text-[var(--text-display-lg)] font-bold leading-tight tracking-tight">
        Operação reputacional em atenção
      </h1>
    </header>
  );
}
```

          Score de risco

          87

## 3. Marque conteúdo assistido pela Cris

```tsx
export function CrisBriefing() {
  return (
    <aside
      className="rounded-md bg-card p-5"
      style={{
        borderLeft: "var(--ai-marker-border)",
        background: "color-mix(in oklab, var(--primary) var(--ai-marker-bg-mix), var(--card))",
      }}
    >
      <p className="font-mono text-xs uppercase tracking-[var(--tracking-kicker)] text-muted-foreground">
        ◐ Escrito com Cris
      </p>
      <p className="mt-3 text-sm leading-relaxed">
        [Inferência] A narrativa está migrando de reclamação pontual para crítica de governança.
      </p>
    </aside>
  );
}
```

## 4. Aplique densidade por superfície

```tsx
<div data-density="compact" className="grid gap-[var(--density-gap)] p-[var(--density-pad-page)]">
  {/* tabelas, filas e operações densas */}
</div>
```

Use `compact` para listas operacionais, `comfortable` como default e `spacious` para leitura ou apresentação executiva.
