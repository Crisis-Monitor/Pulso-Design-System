# Pulso Design System

Design system para **operações de crise, IA auditável e decisão humana**.

Pulso é a linguagem visual do Orchids Crisis Manager: estrutura, tokens,
componentes e padrões para transformar sinais de reputação em leitura operacional
clara, rastreável e acionável.
