# Pulso Design System

Sistema de design para o **Orchids Crisis Manager** — plataforma de gestão de crise e monitoramento de reputação digital.

## Sobre

Pulso é o sistema de design que padroniza a interface visual do Orchids Crisis Manager. Foi criado para garantir:

- **Consistência** entre todas as superfícies do produto
- **Velocidade** no desenvolvimento de novas features
- **Acessibilidade** como requisito, não como exceção
- **Escalabilidade** para crescer com o produto

## Estrutura

| Nível | Descrição | Exemplos |
|-------|-----------|----------|
| **Foundations** | Tokens, cores, tipografia, espaçamento, ícones e motion | Base de todo o sistema |
| **Átomos** | Componentes indivisíveis | Button, Input, Badge, Avatar |
| **Moléculas** | Combinações de átomos | FormField, MetricCard, Accordion |
| **Organismos** | Blocos funcionais completos | Card, Dialog, Table, Tabs |
| **Padrões** | Composições para cenários reais | Crisis Status, Risk Levels |
| **Templates** | Estruturas de página completas | Dashboard, Crisis Detail, Settings |

## Princípios rápidos

1. **Clareza sobre ornamento** — cada elemento existe para comunicar
2. **Ação sobre decoração** — interface como ferramenta, não como vitrine
3. **Dados primeiro** — o conteúdo dita o layout
4. **Status visível** — o usuário sempre sabe o estado do sistema
5. **Consistência contextual** — mesmo padrão, mesma expectativa
