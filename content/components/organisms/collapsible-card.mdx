# Collapsible Card

Card com header sempre visível e conteúdo expansível/colapsável.

## Import

```tsx
import { CollapsibleCard } from "@/components/ui/collapsible-card"
```

## Uso básico

```tsx
<CollapsibleCard title="Configurações avançadas">
  <p>Conteúdo expansível aqui.</p>
</CollapsibleCard>
```

## Inicialmente aberto

```tsx
<CollapsibleCard title="Detalhes da crise" defaultOpen>
  <p>Conteúdo visível ao carregar.</p>
</CollapsibleCard>
```

## Quando usar

- Seções de detalhes que podem ser ocultadas
- Configurações avançadas opcionais
- Logs e informações técnicas
