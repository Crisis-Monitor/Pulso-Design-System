# Empty State

Componente que ocupa o espaço quando não há dados para exibir. Inclui ilustração, texto explicativo e call-to-action.

## Import

```tsx
import { Empty } from "@/components/ui/empty"
```

## Uso básico

```tsx
<Empty
  title="Nenhuma crise encontrada"
  description="Não há crises ativas no momento. Crie uma nova ou ajuste os filtros."
/>
```

## Com ação

```tsx
import { Plus } from "lucide-react"

<Empty
  title="Nenhuma métrica configurada"
  description="Adicione métricas para começar a monitorar."
  action={
    <Button><Plus className="size-4" /> Adicionar métrica</Button>
  }
/>
```

## Quando usar

- Listas vazias
- Resultados de busca sem matches
- Dashboards sem dados configurados

## Quando não usar

- Para erros de carregamento (use Alert)
- Para loading (use Skeleton ou Spinner)
