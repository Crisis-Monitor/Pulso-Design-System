# Button Group

Agrupa botões visualmente conectados para ações relacionadas, como toggles de visualização ou seletores de período.

## Import

```tsx
import { ButtonGroup, ButtonGroupItem } from "@/components/ui/button-group"
```

## Uso básico

```tsx
<ButtonGroup>
  <ButtonGroupItem>Diário</ButtonGroupItem>
  <ButtonGroupItem variant="active">Semanal</ButtonGroupItem>
  <ButtonGroupItem>Mensal</ButtonGroupItem>
</ButtonGroup>
```

## Quando usar

- Toggles de visualização (lista/grid/tabela)
- Seletores de período (dia/semana/mês)
- Ações mutuamente exclusivas
