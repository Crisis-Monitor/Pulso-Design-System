# Input Group

Wrapper para inputs com elementos visuais anexos.

## Import

```tsx
import { InputGroup, InputGroupText, InputGroupButton } from "@/components/ui/input-group"
```

## Com ícone

```tsx
<InputGroup>
  <SearchIcon className="size-4 text-muted-foreground" />
  <Input placeholder="Buscar crises..." />
</InputGroup>
```

## Com botão

```tsx
<InputGroup>
  <Input placeholder="URL do webhook" />
  <InputGroupButton><Button variant="ghost">Testar</Button></InputGroupButton>
</InputGroup>
```

## Quando usar

- Campos de busca com ícone
- Inputs com ações secundárias
- URLs com prefixo fixo
