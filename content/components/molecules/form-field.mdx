# Form Field

Wrapper que organiza label, campo de entrada, descrição e mensagem de erro.

## Import

```tsx
import { Field } from "@/components/ui/field"
```

## Uso básico

```tsx
<Field
  label="Nome da crise"
  description="Um nome descritivo para identificar a crise"
>
  <Input placeholder="Ex: Incidente servidor principal" />
</Field>
```

## Com erro

```tsx
<Field label="Email" error="Email inválido">
  <Input type="email" aria-invalid="true" />
</Field>
```

## Quando usar

- Campos de formulário com label e descrição
- Para manter consistência visual em formulários

## Acessibilidade

- Label conectado ao input via `htmlFor`
- Mensagem de erro conectada via `aria-describedby`
