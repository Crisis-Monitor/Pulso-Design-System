# Avatar Group

Exibe múltiplos avatares sobrepostos, ideal para mostrar equipes, participantes de crise ou responsáveis.

## Import

```tsx
import { AvatarGroup } from "@/components/ui/avatar-group"
```

## Uso básico

```tsx
<AvatarGroup>
  <Avatar><AvatarImage src="/avatars/ana.png" /><AvatarFallback>AN</AvatarFallback></Avatar>
  <Avatar><AvatarImage src="/avatars/bruno.png" /><AvatarFallback>BR</AvatarFallback></Avatar>
  <Avatar><AvatarImage src="/avatars/carlo.png" /><AvatarFallback>CA</AvatarFallback></Avatar>
</AvatarGroup>
```

## Com contador de excedentes

```tsx
<AvatarGroup max={3}>
  {/* 5 avatares — mostra 3 + "+2" */}
  <Avatar>...</Avatar>
  <Avatar>...</Avatar>
  <Avatar>...</Avatar>
  <Avatar>...</Avatar>
  <Avatar>...</Avatar>
</AvatarGroup>
```

## Quando usar

- Mostrar membros de equipe em uma crise
- Participantes de um grupo ou projeto
