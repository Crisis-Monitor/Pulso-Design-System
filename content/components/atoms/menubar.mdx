# Menubar

> **Atom**
>
> Menubar organiza comandos globais de aplicação. Use quando há modelo desktop claro, não como navegação comum.

### React

```tsx
import * as React from "react"

type Menu = { label: string; items: string[] }

export function PulsoMenubar({ menus }: { menus: Menu[] }) {
  return (
    <nav aria-label="Menu da aplicação" className="inline-flex rounded-[var(--radius-md)] border border-[var(--border)] bg-[var(--card)] p-1">
      {menus.map((menu) => (
        <button key={menu.label} type="button" className="rounded-[var(--radius-sm)] px-3 py-1.5 text-sm text-[var(--fg-muted)] hover:bg-[var(--muted)] hover:text-[var(--fg)] focus-visible:outline focus-visible:outline-[var(--ring-width)] focus-visible:outline-offset-2 focus-visible:outline-[var(--ring)]">
          {menu.label}
        </button>
      ))}
    </nav>
  )
}
```

```tsx
<PulsoMenubar menus={[{ label: "Caso", items: ["Arquivar", "Escalar"] }, { label: "Exibir", items: ["Lista", "Timeline"] }]} />
```

```html
<nav class="menubar" aria-label="Menu da aplicação">
  <button type="button">Caso</button>
  <button type="button">Exibir</button>
  <button type="button">Ajuda</button>
</nav>
```

## Regras

1. Menubar é para comandos, não para sidebar ou tabs.
2. Precisa funcionar por teclado.
3. Comandos destrutivos dentro de menus exigem confirmação.
4. Não use se uma navegação simples resolve.
