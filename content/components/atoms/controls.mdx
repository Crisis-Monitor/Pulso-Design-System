# Controls

Container para agrupar controles de formulário com espaçamento e alinhamento consistentes.

## Import

\`\`\`tsx
import { Controls } from "@/components/ui/controls"
\`\`\`

## Uso básico

\`\`\`tsx
<Controls>
  <Input placeholder="Buscar..." />
  <Select><SelectTrigger><SelectValue placeholder="Filtrar" /></SelectTrigger></Select>
  <Button>Aplicar</Button>
</Controls>
\`\`\`

## Quando usar

- Toolbars de filtro e busca
- Grupos de ação em formulários
