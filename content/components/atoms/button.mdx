# Button

Botão comunica ação. Use para submissão, navegação, confirmação ou ação contextual. Em telas operacionais, o texto precisa deixar claro **o que acontece depois do clique**.

## Blueprint React copiável

Este repo documenta o Pulso DS; ele ainda não publica uma biblioteca runtime. Use o exemplo abaixo como blueprint de implementação no produto.

```tsx
import * as React from "react";

type ButtonVariant = "primary" | "secondary" | "ghost" | "destructive";
type ButtonSize = "sm" | "md" | "lg" | "icon";

type PulsoButtonProps = React.ButtonHTMLAttributes<HTMLButtonElement> & {
  variant?: ButtonVariant;
  size?: ButtonSize;
  isLoading?: boolean;
};

const variantClass: Record<ButtonVariant, string> = {
  primary: "bg-[var(--primary)] text-[var(--primary-foreground)] hover:bg-[var(--primary-hover)]",
  secondary: "border border-[var(--border-strong)] bg-[var(--card)] text-[var(--fg)] hover:bg-[var(--muted)]",
  ghost: "bg-transparent text-[var(--fg-muted)] hover:bg-[var(--muted)] hover:text-[var(--fg)]",
  destructive: "bg-[var(--destructive)] text-[var(--destructive-foreground)] hover:opacity-90",
};

const sizeClass: Record<ButtonSize, string> = {
  sm: "h-8 px-3 text-xs",
  md: "h-9 px-4 text-sm",
  lg: "h-10 px-5 text-sm",
  icon: "h-9 w-9 p-0",
};

export function PulsoButton({
  variant = "primary",
  size = "md",
  isLoading = false,
  disabled,
  className,
  children,
  ...props
}: PulsoButtonProps) {
  return (
    <button
      type="button"
      disabled={disabled || isLoading}
      aria-busy={isLoading || undefined}
      className={[
        "inline-flex items-center justify-center gap-2 rounded-[var(--radius-md)] font-medium",
        "transition-colors duration-[var(--motion-duration-fast)] ease-[var(--motion-easing)]",
        "focus-visible:outline focus-visible:outline-[var(--ring-width)] focus-visible:outline-offset-2 focus-visible:outline-[var(--ring)]",
        "disabled:cursor-not-allowed disabled:opacity-60",
        variantClass[variant],
        sizeClass[size],
        className,
      ].filter(Boolean).join(" ")}
      {...props}
    >
      {isLoading ? <span aria-hidden="true">Carregando…</span> : children}
    </button>
  );
}
```

## Uso

```tsx
<PulsoButton onClick={acknowledgeAlert}>
  Reconhecer alerta
</PulsoButton>

<PulsoButton variant="secondary">
  Ver evidências
</PulsoButton>

<PulsoButton variant="destructive">
  Encerrar caso
</PulsoButton>
```

## Regras

- Uma ação primária por superfície.
- Use verbos claros: `Reconhecer alerta`, `Ver evidências`, `Encerrar caso`.
- Não use cor como único indicador de risco; o texto precisa indicar consequência.
- Use `isLoading` apenas quando há espera real.
- Botões destrutivos devem deixar a consequência explícita.
