# Kicker

Texto curto e estilizado que aparece acima de títulos para fornecer contexto ou categorização.

## Import

\`\`\`tsx
import { Kicker } from "@/components/ui/kicker"
\`\`\`

## Uso básico

\`\`\`tsx
<Kicker>Monitoramento</Kicker>
<h2>Crise de segurança detectada</h2>
\`\`\`

## Em cards

\`\`\`tsx
<div className="p-6 border rounded-lg">
  <Kicker>Nova crise</Kicker>
  <h3 className="text-lg font-semibold">Incidente no servidor principal</h3>
</div>
\`\`\`

## Quando usar

- Categorizar seções de conteúdo
- Identificar o tipo de card ou bloco
- Pré-títulos em métricas e alertas
