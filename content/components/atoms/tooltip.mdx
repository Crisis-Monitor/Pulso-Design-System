# Tooltip

Revela informações adicionais quando o usuário passa o mouse ou foca em um elemento.

## Import

\`\`\`tsx
import { Tooltip, TooltipContent, TooltipProvider, TooltipTrigger } from "@/components/ui/tooltip"
\`\`\`

## Uso básico

\`\`\`tsx
<TooltipProvider>
  <Tooltip>
    <TooltipTrigger><button>Hover me</button></TooltipTrigger>
    <TooltipContent><p>Informação adicional aqui</p></TooltipContent>
  </Tooltip>
</TooltipProvider>
\`\`\`

## Posições

\`\`\`tsx
<TooltipContent side="top">Top</TooltipContent>
<TooltipContent side="right">Right</TooltipContent>
<TooltipContent side="bottom">Bottom</TooltipContent>
<TooltipContent side="left">Left</TooltipContent>
\`\`\`

## Acessibilidade

- Acessível via hover E focus
- Delay padrão de 700ms para evitar flash
- Fecha com ESC
- \`TooltipProvider\` é obrigatório no parent
