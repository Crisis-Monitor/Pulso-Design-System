# Label

Rótulo semântico para campos de formulário. Conecta-se ao input via \`htmlFor\`.

## Import

```tsx
import { Label } from "@/components/ui/label"
```

## Uso básico

```tsx
<Label htmlFor="email">Email</Label>
<Input id="email" type="email" />
```

## Acessibilidade

- \`htmlFor\` deve corresponder ao \`id\` do input
- Clique no label foca o input automaticamente
