# Chip

Elemento inline compacto usado para filtros, tags selecionáveis e categorias.

## Import

```tsx
import { Chip } from "@/components/ui/chip"
```

## Uso básico

```tsx
<Chip>Monitoramento</Chip>
```

## Variantes

| Variante | Uso |
|----------|-----|
| `default` | Tag neutra, filtro padrão |
| `outline` | Filtro não selecionado |

## Removível

```tsx
<Chip onClose={() => console.log("removed")}>Crise ativa</Chip>
```

## Como filtro

```tsx
<div className="flex flex-wrap gap-2">
  <Chip variant="default">Todas</Chip>
  <Chip variant="outline">Ativas</Chip>
  <Chip variant="outline">Resolvidas</Chip>
</div>
```

## Quando usar

- Filtros em dashboards e listas
- Tags selecionáveis
- Categorias com estado ativo/inativo
