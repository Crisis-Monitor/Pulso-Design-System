# Live Indicator

Indicador visual que mostra se um dado ou monitoramento está ativo e em tempo real. Usa um ponto pulsante animado.

## Import

\`\`\`tsx
import { LiveIndicator } from "@/components/ui/live-indicator"
\`\`\`

## Uso básico

\`\`\`tsx
<LiveIndicator label="Ao vivo" />
\`\`\`

## Estados

\`\`\`tsx
<LiveIndicator status="live" label="Ao vivo" />
<LiveIndicator status="offline" label="Offline" />
<LiveIndicator status="connecting" label="Conectando..." />
\`\`\`

## Quando usar

- Mostrar que dados estão sendo atualizados em tempo real
- Status de conexão com serviços
- Indicador de monitoramento ativo
