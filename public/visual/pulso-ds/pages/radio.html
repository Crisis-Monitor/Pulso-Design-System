Atoms · Onda 3 · Form 02

Radio

Escolha exclusiva. Diferente do Checkbox no formato (círculo) e na intenção: exatamente uma opção do grupo. Quando > 5 opções ou opções têm hierarquia, prefira Select.

Estados

Mesma calibragem do Checkbox: 16px padrão, motion sutil ao marcar (160ms scale com bounce).

Grupo com descrição

Cada opção descreve sua consequência operacional. Spacing 14px entre opções para leitura confortável.

Card variant

Para escolhas com peso operacional — modos de operação, planos, fluxos. Use 2–4 opções; mais que isso prejudica comparação.

Segmented control

Radio renderizado como toggle inline. Padrão para 2–4 opções curtas — período, modo de visualização, escala. Ocupa pouco espaço; ideal em barras de filtro e headers.

Quando usar

Use
  • 2–5 opções mutuamente exclusivas
  • Quando todas as opções precisam ficar visíveis para comparação
  • Segmented para escolhas curtas e frequentes (período, modo)
Evite
  • Mais de 5 opções → use Select
  • Toggle on/off → use Switch
  • Múltipla escolha → use Checkbox
