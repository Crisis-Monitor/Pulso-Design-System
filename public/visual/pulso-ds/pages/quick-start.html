Quick Start

Primeira tela em 5 minutos: page header + metric cards + tabela. Os imports já assumem que a instalação foi concluída.

Esqueleto de uma página

import { PageHeader } from '@pulso-ds/core/page-header';
import { MetricCard } from '@pulso-ds/core/metric-card';
import { Table } from '@pulso-ds/core/table';

export default function DashboardPage() {
  return (
    <div className="px-8 py-10 space-y-8">
      <PageHeader
        kicker="Operação · Hoje"
        title="Bom dia, Neviton."
        sub="3 casos pedem revisão antes das 10h."
      />

      <div className="grid grid-cols-4 gap-4">
        <MetricCard tone="violet" label="Menções 24h" value="1.247" delta="+12%" />
        <MetricCard tone="emerald" label="Sentimento" value="+0.38" delta="+0.04" />
        <MetricCard tone="rose" label="Críticos" value="3" delta="+1" />
        <MetricCard tone="amber" label="SLA médio" value="2h 14m" delta="-18m" />
      </div>

      <Table data={cases} columns={columns} />
    </div>
  );
}

Convenções essenciais

  • → Use tokens semânticos (--risk-attention), nunca hex direto.
  • → Números/IDs/timestamps em font-mono (IBM Plex Mono).
  • → Density: data-density="compact|comfortable|spacious" no html.
  • → Cor primária só em ação primária, foco e métricas-chave.
