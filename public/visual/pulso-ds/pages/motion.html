Foundations · 06

Motion

Movimento confirma ação, não chama atenção. Durações curtas (≤200ms) para tudo que é resposta direta. Animação enfática só quando o objeto entra ou sai do espaço.

Durações

--motion-duration-fast120msHover, focus, micro-interação
--motion-duration-default200msTransição de estado, expand/collapse
--motion-duration-slow320msLayout, page transition, modal

Pulse

A única animação contínua: indica atividade ao vivo. Mantida mesmo com prefers-reduced-motion porque é informação, não decoração.

Caso ativo · monitorando

Reduced motion

Respeitamos prefers-reduced-motion: reduce. Toda animação cai para 0.01ms exceto pulse-opacity (sinal vital de live data).

Regras

DO
120ms para hover. 200ms para mudança de estado. 320ms para modal/page.
DO
Use translate/opacity. Evite animar width/height (custo de layout).
DON'T
Não use animação como única forma de feedback — sempre acompanhe com mudança de estado visual.
DON'T
Não anime cores de uma para outra atravessando hue ruim. Use color-mix se precisar.
