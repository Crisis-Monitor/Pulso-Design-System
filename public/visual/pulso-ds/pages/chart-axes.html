Components · Charts · 03

Axes, grids, tooltips, legendas

Os componentes auxiliares ditam 60% da legibilidade. Sem padrão, cada chart parece de uma biblioteca diferente. Esta página fixa os tokens dedicados, o componente ChartTooltip reutilizável e os 4 posicionamentos de legend.

Eixos · X e Y

Ticks mono tabular-nums · labels rotacionados a 0° (nunca 45°/90° em desktop) · alinhamento padronizado para todos os 6 primitives.

Grid

Sugere, não compete. Sempre horizontal (exceto heatmap, que tem grid implícito nas células). Alpha baixo · grossura 1px constante.

✓ Grid canônico
Horizontal apenas · alpha 6% · 1px solid · sem chamar atenção.
✗ Grid pesado
Verticais + alpha alto · briga com os dados. Operador procura o ponto dentro do quadriculado.

Anchor
Acima do ponto por padrão. Se cortar topo do viewport, flip para baixo. Distância 16px do ponto · arrow centralizada.
Trigger
Hover em desktop · touch tap em mobile · sempre escutável via teclado (foco em ponto).
Crosshair
Linha vertical pontilhada do ponto até o eixo X · stroke --fg-subtle · stroke-dasharray 2 3.
Animação
Fade-in 200ms --motion-easing · delay 80ms (evita flicker em scrubbing rápido). Reduced-motion: instantâneo.

Valores via formatNumber() · timestamp via formatDate('datetime'). Ver 2.3 Localização.

Legend · 4 posições

Top/bottom para charts wide · right para charts com muitas séries e altura sobrando · false quando o chart é sparkline ou single-series com endpoint marcado. Clique em item da legend toggla a série (com micro-animação de opacidade).

Anti-padrões

✗ Tooltip cobrindo o ponto
Operador não vê o que está medindo. Posicione acima/abaixo com distância ≥ 16px.
✗ Labels rotacionados a 45°
Indica que os labels não cabem. Solução: truncar (12/05 vs 12 maio), mostrar 1 em cada 2, ou aumentar largura.

Regras

Faça
  • Use ChartTooltip sempre · não escreva tooltip custom por chart.
  • Labels do eixo em mono tabular-nums · rotação 0°.
  • Threshold importante (ex: 75 do escalonamento) sempre marcado em linha dashed + label.
  • Legend top para ≤ 4 séries · right para ≥ 5 · bottom para charts com título acima.
  • Clique em legend toggla a série · dot vira outline quando off.
Evite
  • Não use grid vertical (exceto heatmap).
  • Não rotacione labels 45°/90° em desktop. Trunca ou aumenta o chart.
  • Não posicione tooltip cobrindo o ponto medido.
  • Não use tooltip claro no light theme. Sempre escuro · contraste forte.
  • Não esconda legend quando há ≥ 2 séries. Endpoint label não substitui.
