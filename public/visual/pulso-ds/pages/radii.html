Foundations · 04

Radii

No v2 reduzimos os raios. O v1 tinha 22px em superfícies; agora o teto é 14px. Cantos firmes comunicam confiança e seriedade — exatamente o tom que ferramenta de gestão de crise pede.

Escala

--radius-xs
4px · chips, toggles
--radius-sm
6px · inputs, botões
--radius-md
8px · cards, menus
--radius-lg
10px · superfícies
--radius-xl
14px · feature
--radius-full
9999px · pills, avatar

Focus ring

Anel de foco em todo elemento interativo. Sempre visível para teclado.

Regras

DO
Use radius-md (8px) como default para qualquer container. radius-sm para controles inline.
DO
Mantenha consistência hierárquica: nunca aninhe um radius maior dentro de um menor.
DON'T
Não use radius-full em botões grandes — reservado para chips, badges, avatares.
DON'T
Não misture raios "estranhos" (12, 16) — saia da escala só com motivo claro.
