Form Field

Container que une label + input + helper/erro/sucesso. Padrão para qualquer campo de formulário no Pulso. Label monoespaçada acima, mensagem de feedback abaixo.

Anatomia

Label em mono uppercase 10.5px (tracking 0.08em). Helper/erro em sans 12px abaixo. Spacing fixo 6px entre todos os elementos.

Obrigatório
Visível para todos os membros do time. Pode ser alterado depois.

Estados

Padrão · Erro · Sucesso · Disabled. Mensagem abaixo do campo, ícone à esquerda do texto. Nunca remova label em estado de erro — sempre fornece contexto para correção.

Diretrizes

Faça
  • Helper text para contexto · 1 linha, < 80 caracteres
  • Mensagem de erro indica como corrigir, não só o que está errado
  • Asterisco vermelho discreto para obrigatório
  • Ícone a esquerda da mensagem para reconhecimento rápido
Evite
  • Label dentro do input (esconde após digitar — perde contexto)
  • Mensagens vagas ("Erro", "Inválido")
  • Marcar todos os campos como obrigatórios — marque os opcionais
  • Helper + erro ao mesmo tempo · só um por vez
