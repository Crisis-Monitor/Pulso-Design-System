Tooltip

Texto contextual sobre hover/focus. Use para nomear ícones, mostrar atalhos e explicar abreviações. Não use para conteúdo essencial — quem está em mobile não vê.

Regras

DO
Use Tooltip para nomear ícones isolados, mostrar atalhos e expandir abreviações editoriais (TIER 1, σ, MAU).
DO
Texto curto (≤ 5 palavras) no default. Para mais que isso, use Rich com título.
DO
Auto-flip se o tooltip ficar parcialmente fora do viewport.
DON'T
Não esconda informação essencial em Tooltip — quem usa mobile/teclado não consegue ler.
DON'T
Não use Tooltip em links e botões com label visível. É redundância e cria ruído de hover.
DON'T
Tooltip não é Popover. Não coloque ações clicáveis dentro — não é interativo.
