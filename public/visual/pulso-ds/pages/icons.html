Icons

O Pulso usa Lucide como biblioteca de ícones — line-style, 1.5px, métrica consistente. Tamanhos canônicos: 12, 14, 16, 20. Cor sempre via currentColor.

Tamanhos

Quatro tamanhos. 14px é o default para uso em-linha com texto body.

12 · micro
14 · inline
16 · button
20 · feature

Convenções

Do
currentColor sempre. Stroke 1.5px. Alinhe ao texto via flex + align-items:center.
Don't
Não preencha ícones com fill sólido. Não use família mista (lucide + emoji + custom svg).

Pacote

import { AlertTriangle, Activity, BellRing, ChevronRight } from 'lucide-react';

<AlertTriangle size={14} aria-hidden="true" />
