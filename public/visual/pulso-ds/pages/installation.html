Getting Started · 01

Instalação

Como instalar e configurar o Pulso DS num projeto Next.js + Tailwind 4. Pré-requisito: Node 20+, pnpm/bun.

1. Dependências

pnpm add @pulso-ds/core @pulso-ds/icons
pnpm add tailwindcss@^4 tw-animate-css
pnpm add @radix-ui/react-* class-variance-authority clsx

2. Tokens

Importe globals.css no app/layout.tsx. Os tokens entram via @theme inline e ficam disponíveis como utilities do Tailwind 4.

/* app/globals.css */
@import "tailwindcss";
@import "tw-animate-css";
@import "@pulso-ds/core/globals.css";

@custom-variant dark (&:is(.dark *));

3. Fontes

// app/layout.tsx
import { Manrope, IBM_Plex_Mono } from 'next/font/google';

const manrope = Manrope({ subsets: ['latin'], variable: '--font-manrope' });
const mono = IBM_Plex_Mono({ subsets: ['latin'], weight: ['400','500','600'], variable: '--font-ibm-plex-mono' });

export default function RootLayout({ children }) {
  return (
    <html lang="pt-BR" className={`${manrope.variable} ${mono.variable}`}>
      <body>{children}</body>
    </html>
  );
}

4. Theme provider

O Pulso usa next-themes para light/dark. A classe .dark aplicada no html ativa o tema escuro.

// app/providers.tsx
'use client';
import { ThemeProvider } from 'next-themes';
export function Providers({ children }) {
  return <ThemeProvider attribute="class" defaultTheme="light">{children}</ThemeProvider>;
}
