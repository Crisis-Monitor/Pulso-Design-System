Audit script · como medimos

Roda em CI · uma pasta, três outputs: token coverage, bundle size, perf budget. Tudo dump em artifacts/audit.json; bot comenta no PR.

# package.json · scripts
"audit:tokens": "pulso-audit tokens --root src --report json",
"audit:bundle": "size-limit --json",
"audit:perf":   "node scripts/perf-budget.mjs",
"audit":        "npm-run-all audit:tokens audit:bundle audit:perf"

# .github/workflows/audit.yml
- run: npm run audit
- uses: pulso/audit-action@v1
  with:
    artifact: artifacts/audit.json
    fail-on: coverage<0.95, bundle>80kb, lcp>2s
