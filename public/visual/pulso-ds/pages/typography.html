Pesos

Manrope: 400, 500, 600, 700, 800. Use 800 só em display hero. Nunca em corpo.

400 · Regular — texto de corpo padrão
500 · Medium — labels, ênfase sutil, números mono
600 · Semibold — títulos de seção
700 · Bold — títulos de página, display
800 · Extrabold — apenas hero, marketing

Regras

DO
Use mono para qualquer número técnico — score, contagem, timestamp, ID. Dá ritmo e ancora a leitura.
DO
Use a escala definida. Se algo "pede" um tamanho intermediário, é sintoma de outro problema (provavelmente layout).
DON'T
Não use opacity para criar hierarquia. Use os tokens fg/fg-secondary/fg-muted/fg-subtle.
DON'T
Não misture sans e mono em uma mesma frase. Mono é para tokens isolados (números, IDs).
