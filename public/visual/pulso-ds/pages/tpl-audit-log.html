Edge cases

  • Evento de sistema (sem actor humano) · avatar do Pulso IA · role "sistema" · IP/UA = API origin.
  • Filtro > 50k eventos · banner "Refine o filtro · ou exporte para CSV" · paginação suspende após 10k carregados.
  • Evento > 90 dias · não aparece no grid online · disponível via export "histórico arquivado" (acionável por gestor apenas).
  • Operador olhando audit · vê só os próprios eventos · gestor + analista veem tudo do workspace · acesso cross-workspace só com SSO de auditoria.
