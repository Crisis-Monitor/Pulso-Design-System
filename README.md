# Pulso Design System

Design System do projeto Crisis Monitor — gestão de crises e monitoramento de mídia.

## Getting Started

```bash
bun install
bun dev    # http://localhost:3002
bun run build
```

## Tech Stack

- Next.js 15 + Nextra
- Tailwind CSS v4
- shadcn/ui components
- Lucide icons

## Contribuindo

1. Crie uma branch a partir de `main`
2. Implemente a mudança
3. Abra um Pull Request
